项目概述

本项目旨在构建一套完整的智能化文档管理系统，通过AI技术提升文档创建、编辑和管理的效率。

系统采用前后端分离架构，前端使用 Next.js + React，后端使用 Python FastAPI。

技术架构

前端技术栈包括：

  • Next.js 16 (App Router)
  • React 19
  • Tailwind CSS 4

后端采用 LangGraph + FastAPI 构建，支持流式响应和工具调用。

交互流程

默认状态 编辑器占满空间，底部显示一行 AI 输入栏 + 快捷操作按钮 + 发送按钮
展开对话 用户发送消息后，AI 对话区域展开（编辑器缩小），显示多轮对话
快捷操作 收起时在底部输入栏旁显示；展开时在对话 Header 中显示
替换/复制 每条 AI 回复下方有「替换」和「复制」按钮
收起对话 点击 ▼ 按钮收起对话区域，编辑器恢复全高
选中文字 编辑器中选中文字后，点击快捷操作或输入指令，选中文字自动附带到消息中

设计优势

  • 编辑器保持全宽，不被压缩
  • AI 对话内嵌在编辑器内，体验一体化
  • 展开/收起切换，灵活控制空间
  • 与千问 AI 笔记交互模式一致

实现要点

  • 对话区域高度可动态调整（拖拽或固定比例）
  • 展开时编辑器可滚动查看内容
  • 收起/展开有过渡动画
