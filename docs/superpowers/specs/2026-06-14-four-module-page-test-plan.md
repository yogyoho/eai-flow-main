- **AI 生成依赖 LLM**: t1-ai 节点需 LLM 配置可用；若超时/失败属环境问题，记为 BLOCKED 而非功能缺陷
- **zhangsan 双重身份**: 系统角色=部门负责人，项目角色=组员；审核环节需观察实际绑定逻辑
- **Temporal 工作流引擎**: 节点推进依赖 temporal 服务（eai-flow-temporal 容器健康）
