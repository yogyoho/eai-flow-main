## 测试流程场景

**场景**: 消防设计专篇报告 — 从角色创建到文档发布的完整流程

```
1. [超级管理员] 创建角色 → 创建部门 → 分配用户
2. [项目经理] 创建工作流模板 → 编排消防设计专篇流程
3. [项目经理] 创建项目 Card → 配置团队 → 设置章节
4. [撰写人] AI 生成初稿 → 人工编辑 → 保存版本
5. [审核人] 审核章节 → 添加批注 → 通过/退回
6. [项目经理] 提交审批 → 跟踪进度
7. [部门负责人] 审批报告 → 通过/驳回
8. [项目经理] 导出 Word → 发布文档
```
