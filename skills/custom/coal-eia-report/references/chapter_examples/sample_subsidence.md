## X.1 矿区开采地表沉陷预测

### X.1.1 沉陷预测模型

地表沉陷预测采用**概率积分法**模型。该模型基于随机介质理论，适用于煤层开采引起的地表移动变形预测。

**（1）稳定态预计模型**

在倾斜煤层中开采某单元i，按概率积分法的基本原理，单元开采引起地表任意点(x，y)的下沉(最终值)为：

```
We0i(x,y) = (1/r²)·exp(-π(x-xi)²/r²)·exp(-π(y-yi+li)²/r²)
```

式中主要参数：
- r — 主要影响半径，r = H₀/tanβ
- H₀ — 平均采深
- tanβ — 预计参数，为主要影响角β之正切
- li = Hi·cotθ，θ为最大下沉角
- (xi, yi) — 单元中心点坐标； (x, y) — 地表任意点坐标

**（2）主要变形指标计算公式**

设工作面范围为走向长p、倾向水平距离a组成的矩形：

| 变形指标 | 计算公式 | 说明 |
|---------|---------|------|
| 地表下沉 W(x,y) | W₀×∫∫Weoi(x,y)dxdy | W₀ = mqcosα，q为下沉系数 |
| 倾斜 i(x,y,φ) | 对W求φ方向方向导数 | φ为预计方向与x轴夹角 |
| 曲率 k(x,y,φ) | 对i求φ方向方向导数 | 反映地表弯曲程度 |
| 水平移动 U(x,y,φ) | 含走向和倾向分量 | 方向性指标 |
| 水平变形 ε(x,y,φ) | 含拉伸和压缩分量 | 评价建筑物影响的关键指标 |

**（3）充分采动时最大值预计**

| 指标 | 公式 |
|------|------|
| 最大下沉值 | W₀ = mqcosα |
| 最大倾斜值 | i₀ = W₀/r |
| 最大水平移动 | U₀ = bW₀ |
| 最大曲率值 | k₀ = 1.52·W₀/r² |
| 最大水平变形 | ε₀ = 1.52·b·W₀/r |

### X.1.2 参数选取

地表移动变形计算的主要输入参数包括：

| 参数 | 符号 | 含义 | 影响因素 |
|------|------|------|---------|
| 下沉系数 | q | 反映地表最大下沉与采厚关系 | 开采方法、顶板管理、覆岩性质、重复采动 |
| 主要影响角正切 | tanβ | 反映开采影响范围 | 覆岩性质、采深采厚比 |
| 水平移动系数 | b | 水平移动与下沉比值 | 覆岩性质 |
| 拐点移动距 | S | 沉陷盆地拐点偏移量 | 开采深度、覆岩性质 |
| 影响传播角 | θ | 反映下沉传播方向 | 煤层倾角、覆岩性质 |

**参数取值方法**：参考矿区已有矿井地表岩移观测结果，结合各井田煤层赋存条件确定。对于无实测数据的新建矿井，可类比地质条件相似的邻近矿井参数。

### X.1.4 沉陷表现形式及特点

根据地表沉陷影响预测结果，矿区地表沉陷变形特点一般包括：

（1）下沉是逐步形成的，要经历较长的时间；
（2）开采下沉造成地形坡度变化只发生在采空区边界上方，是局部区域；
（3）地表沉陷对区域地表形态和自然景观的影响程度取决于下沉值与地表落差的比例关系；
（4）煤层埋深较小的区域，深厚比小于约30时，地表多表现为裂缝、台阶等非连续变形；埋深较大的区域，沉陷一般为整体缓慢下沉；
（5）干旱少雨地区且浅部含水层不发育时，沉陷一般不会导致地表形成积水区。

### X.1.5 保护煤柱留设

煤柱留设宽度按埋深最大的煤层埋深计算，采用垂直剖面法留设保护煤柱。

**主要保护目标类型**：
- 高速公路、铁路 — 按相关规范留设保护煤柱
- 高压输电线路 — 根据电压等级确定保护等级
- 水库及大坝 — 留设禁采区
- 文物保护单位 — 按保护范围外扩安全距离留设
- 自然保护区 — 按实验区边界外扩维护带宽度留设
- 工业园区及重要基础设施 — 按建筑物保护等级留设

**结论表述模式**：根据本次地表沉陷预测结果，[保护目标]位于沉陷影响范围[外/边缘/内]，[会/不会]受到开采沉陷影响。为保证[保护目标]安全，本次评价提出对其留设[XX-XX]m的保护煤柱。
