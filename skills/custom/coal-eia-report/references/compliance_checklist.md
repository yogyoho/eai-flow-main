# 煤炭建设项目环境影响评价报告 合规性审查清单

> 本清单用于验证 EIA 报告是否符合中国现行法律法规、技术导则、标准规范的要求。
> 审查人应逐项确认，在对应条目打勾 `- [x]` 表示已通过。

---

## 1. 法律法规层

- [ ] **《中华人民共和国环境保护法》**（2014年修订）：报告是否引用并在评价中体现"预防为主、综合治理"原则
- [ ] **《中华人民共和国环境影响评价法》**（2018年修正）：报告编制程序是否符合编制—技术评估—审批流程，是否附具公众参与说明
- [ ] **《中华人民共和国煤炭法》**（2016年修正）：是否引用煤矿建设相关环保条款，是否涉及煤炭资源开发许可
- [ ] **《建设项目环境保护管理条例》**（国务院令第682号，2017年修订）：报告分类是否正确（报告书/报告表），是否包含环境影响评价文件审批要求
- [ ] **《中华人民共和国土壤污染防治法》**（2019年施行）：是否涉及土壤环境质量调查、土壤污染防治措施及土壤环境质量标准引用

---

## 2. 技术导则层

- [ ] **HJ 130-2019《规划环境影响评价技术导则 煤炭工业矿区总体规划》**：矿区总体规划环评是否按此导则编制，评价内容是否完整覆盖
- [ ] **HJ 463-2009《规划环境影响评价技术导则 煤炭工业矿区总体规划》**（补充）：是否参照早期版本中仍适用的方法学要求（如有）
- [ ] **HJ 2.2-2018《环境影响评价技术导则 大气环境》**：大气评价等级判定是否正确，AERSCREEN/AERMOD 模型选用是否合规，气象数据来源是否可追溯
- [ ] **HJ 2.3-2018《环境影响评价技术导则 地表水环境》**：地表水评价等级和范围是否正确，预测模型选取是否与导则一致
- [ ] **HJ 610-2016《环境影响评价技术导则 地下水环境》**：地下水评价分级是否正确，水文地质参数是否有据可查
- [ ] **HJ 2.4-2009《环境影响评价技术导则 声环境》**：噪声评价范围是否满足导则要求（200m），预测模式是否正确选用
- [ ] **HJ 19-2011《环境影响评价技术导则 生态影响》**：生态评价等级和范围是否正确，生态调查方法是否按导则执行
- [ ] **HJ 964-2018《环境影响评价技术导则 土壤环境（试行）》**：土壤评价等级是否正确，土壤环境质量标准选用是否适当
- [ ] **HJ 619-2011《环境影响评价技术导则 煤炭采掘工程》**：是否按此导则开展沉陷预测、矸石处置、矿井水处理等专题评价

---

## 3. 标准规范层

- [ ] **GB 3095《环境空气质量标准》**：适用标准类别（一类/二类）判定是否正确，评价因子及浓度限值引用是否准确
- [ ] **GB 3838《地表水环境质量标准》**：水功能区划是否与地方水功能区划一致，标准值引用是否正确
- [ ] **GB/T 14848《地下水质量标准》**：地下水质量分类与项目所在地地下水功能区划是否匹配
- [ ] **GB 3096《声环境质量标准》**：声环境功能区类别判定是否正确，对应限值引用是否准确
- [ ] **GB 15618《土壤环境质量 农用地土壤污染风险管控标准》/ GB 36600《土壤环境质量 建设用地土壤污染风险管控标准》**：土壤标准选用是否与用地类型一致（农用地 vs 建设用地）
- [ ] **GB 12348《工业企业厂界环境噪声排放标准》**：厂界噪声评价标准是否正确选用，是否区分昼夜标准值
- [ ] **GB 16297《大气污染物综合排放标准》**：大气污染物排放浓度限值和速率限值引用是否准确，是否考虑排气筒高度折算
- [ ] **GB 20426《煤炭工业污染物排放标准》**：煤炭行业特有污染物（颗粒物、SO₂ 等）排放限值是否优先引用本标准

---

## 4. 评价方法层

- [ ] **沉陷预测方法**：是否采用概率积分法（或其他经论证的方法），参数取值是否有实测或类比依据，沉陷范围预测结果是否制图
- [ ] **大气模型选取**：是否按 HJ 2.2-2018 判定评价等级后选取 AERSCREEN（三级）/ AERMOD（二级及以上）模型，模型输入参数是否完整
- [ ] **地表水模型选取**：是否根据 HJ 2.3-2018 选取合适的预测模型（解析解/数值解），参数率定是否有依据
- [ ] **地下水评价方法**：是否按 HJ 610-2016 要求采用解析法或数值法，水文地质概念模型是否合理
- [ ] **噪声预测方法**：是否按 HJ 2.4-2009 选用工业噪声/交通噪声预测模式，声源参数是否有实测或设备资料支撑
- [ ] **生态调查方法**：是否按 HJ 19-2011 布设样方/样线，调查时间是否覆盖植物生长期，生物多样性指数计算是否规范
- [ ] **清洁生产分析**：是否对照《煤炭行业清洁生产评价指标体系》进行分级评定（一级/二级/三级），指标覆盖是否全面
- [ ] **循环经济分析**：是否分析煤矸石综合利用、矿井水回用、瓦斯抽采利用等循环经济链条
- [ ] **公众参与**：是否按规定开展两次信息公开，是否包含公众意见调查表、座谈会记录等，意见采纳情况是否逐条回应
- [ ] **环境监测计划**：是否包含施工期和运营期监测方案，监测点位布设是否合理，监测因子是否覆盖主要环境影响因子
- [ ] **跟踪评价方案**：是否按 HJ 619-2011 设置跟踪评价内容，是否明确跟踪评价的时段、指标和方法

---

## 5. 数据完整性层

- [ ] **图件完整性**：报告是否包含必要的附图：地理位置图、总平面布置图、评价范围图、敏感目标分布图、沉陷等值线图、监测点位图等
- [ ] **监测数据可溯源性**：现状监测数据是否注明监测单位、监测时间、监测方法，引用的历史数据是否注明来源和时效性
- [ ] **计算输入参数可查性**：预测计算所用参数（源强、气象、水文等）是否逐一列出，取值依据是否可查（实测/类比/规范取值）
- [ ] **环保措施可追溯性**：各项环保措施是否与影响预测结果对应，措施的技术可行性和经济合理性是否有论证
- [ ] **附件完整性**：是否附具：建设项目立项文件、环境影响评价委托书、监测报告、公众参与说明、相关规划文件等
